Grid

This example demonstrates a 50%/50% split on small, medium and large devices. On extra small devices, it will stack (100% width).

Resize the browser window to see the effect.

Bootstrap's grid system allows up to 12 columns across the page. If you do not want to use all 12 column individually, you can group the columns together to create wider columns.
Rows must be placed within a .container (fixed-width) or .container-fluid (full-width) for proper alignment and padding Use rows to create horizontal groups of columns
